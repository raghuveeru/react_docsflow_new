# Momster COS App

## Development and running on your machine

````
git clone https://github.com/PebbleRoad/react-momster-workflow-app.git
cd react-momster-workflow-app
npm install
````

## Run a development server

````
npm run start
````

## Building bundle

````
npm run build
````
